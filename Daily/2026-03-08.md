### Block 1 — Discrete Math

Topic: modular arithmetic and polynomial divisibility.

Goals:

- Goal 1 (10 min): prove the modular arithmetic rules.
  Show that if $a \equiv b \pmod m$ and $c \equiv d \pmod m$, then:
  1.  $a + c \equiv b + d \pmod m$
  2.  $ac \equiv bd \pmod m$
      Use the divisibility definition $m \mid (a-b)$.
- Goal 2 (10 min): prove $n^3 - n$ is divisible by $6$ for all integers $n$ using factorization.
- Goal 3 (15 min): prove $n^3 - n$ is divisible by $6$ again using modular arithmetic.
  Show:
  1.  $n^3 - n \equiv 0 \pmod 2$
  2.  $n^3 - n \equiv 0 \pmod 3$
- Goal 4 (10 min): prove $n^5 - n$ is divisible by $30$ for all integers $n$.
  Check divisibility by $2$, $3$, and $5$.

Done when:

### Block 2 — Backprop Shape Intuition

Goal: trace forward and backward tensor shapes in a 2-layer neural network.

Network:

```text
input -> Linear -> ReLU -> Linear -> loss
```

Setup:

```text
batch = 32
input_dim = 10
hidden = 6
output = 1
```

Forward shapes:

```text
X   (32 x 10)
W1  (10 x 6)
b1  (6)

Z1 = XW1 + b1  -> (32 x 6)
A1 = ReLU(Z1)  -> (32 x 6)

W2  (6 x 1)
b2  (1)

Z2 = A1W2 + b2 -> (32 x 1)
loss           -> scalar
```

Backward check:

```text
dZ2
dW2
db2
dA1
dZ1
dW1
db1
```

Example:

```text
dW2 = A1^T dZ2 -> (6 x 32)(32 x 1) = (6 x 1)
```

Expected final shapes:

```text
dZ2 (32 x 1)
dW2 (6 x 1)
db2 (1)

dA1 (32 x 6)
dZ1 (32 x 6)

dW1 (10 x 6)
db1 (6)
```

Done when:

- You can explain why each gradient has its shape without guessing.
